# The Shy Dock

A tiny macOS menu bar app that auto-hides your Dock when you're on the laptop alone and brings it back when an external monitor is connected.

**Website:** [projects.byvova.com/the-shy-dock](https://projects.byvova.com/the-shy-dock)

**Download:** [Releases](https://github.com/vovapyc/The-Shy-Dock/releases)
